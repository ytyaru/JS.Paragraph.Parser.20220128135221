# 強調表示

　小説における傍点（圏点）。

中間形式
```
ここは＊慎重に＊行動すべきだ。
```
HTML
```html
<p>ここは<em>慎重に</em>行動すべきだ。</p>
```
CSS
```css
em {
    text-emphasis-style: filled; /* 横=●縦=﹅ */
}
```
